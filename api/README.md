# Backend API

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Set environment variables (optional):

   - `PORT` (default: 3001)
   - `MONGO_URI` (default: mongodb://localhost:27017/crawler)

3. Start the server:
   ```bash
   node index.js
   ```
